# Customer Intelligence Platform (Unsupervised Learning)

## Overview
This project builds an end-to-end customer analytics pipeline using unsupervised learning techniques.
It focuses on behavioral feature engineering, customer segmentation, and analytics-ready outputs for dashboards.

## Key Features
- Transaction-level data cleaning
- Behavioral feature engineering (basket size, purchase rhythm, product diversity)
- Unsupervised customer segmentation (planned)
- Dashboard-ready outputs (Power BI compatible)

## Tech Stack
- Python
- Pandas, NumPy, Scikit-learn
- Power BI (for visualization)
